# Pixkit Documentation

PIXKIT是全球滑板底盘先驱PIX Moving打造的一站式开源自动驾驶开发教学套件，依托企业核心技术积淀与全球化研发生态，为全球开发者、高校及科研机构搭建低成本、高效率的自动驾驶开发平台，彻底打破行业技术壁垒，大幅降低开发准入门槛，成为Autoware基金会官方推荐车辆平台。
